About Us

The Project Automatic Cheque Settlement System is the Fastest Way of Clearing of Cheques In this The Cheque Will First Scanning by Some Algorithms Model to Find or Clear cut the the A/C Number and all details which needs to be requires for Transferring the money from one bank to another bank account. The whole detail will store in database and throughout on the server the payment request will hold until the process gets success. The Whole process will take less than an hour or depend on server speed since the main thing i.e. to clear out the details from cheque using latest technologies like OpenCV, ML library, Front-End, Databases, Networks, etc. This System would be Helpful for Customers and Save lot of Time.
